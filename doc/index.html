Interval:__le (b)
Compare Less equal(b) '<='. The definition of 'less or equal' follows the definition provided by Bohlender, Kulisch and Lohner 2008. Take two intervals A = [a1,a2] and B = [b1,b2]. A is said to be less or equal B if A <= B : a1 <= b1 and a2 <= b2 To say explicit: A <= B is totally different from (A < B or A == B) !

Parameters:

  • b #Interval to be compared

Returns:

    true if self <= b

Usage:

    if iv1 <= iv2 then ...
if iv1 >= iv2 then ...

Interval:compare (b)
Compare intervals. Compares the relation between self (a) and another interval/value (b)

Parameters:

  • b #Interval or b #number to be checked

Returns:

    #string i, #string v first string reflects the relation of the intervals:
    • '<' if a is less than b
    • '==' if intervals are equal
    • 'a[b]' if b is fully included in a
    • 'b[a]' if a is included in b
    • '<=' if a and b have common elements and a.low <= b.low
    • '=>' if a and b have common elements and a.high >= b.high
    • '>' if a is greater than b
    • '?' in any other case (would be an implementation fault)
    The second string reflects the relation of the values:
    • '<' a.value < b.value
    • '==' if a.value == b.value
    • '>' a.value > b.value

String conversion. Converts the Interval to a string This function calls Interval:format with the standard format.

Interval:format (f)
String format. Formats the Interval to a string with a given format and unit.

Parameters:

  • f Format string. In the format string can be the following tokens that are substituted by the fields of the Interval:
    • '#m' Middle, substituted by self.value
    • '#l' Low, substituted by self.low
    • '#h' High, substituted by self.high
    all tokens will be replaced by a '%' character so that the following characters define the format string for the string.format() function. So the format string '#m5.1f [#l5.1f,#h5.1f]' will do the same like string.format('%5.1f [%5.1f,%5.1f]', iv.value, iv.low, iv.high)

Returns:

    String according to given format.

Usage:

     a = Interval:new{l=1.5,v=2,h=2.5}
 a:format('Result: #m5.1f [#lf,#hf]')
   gives 'Result:   2.0 [1.5,2.5]'
Interval:toJson ()
Generate JSon string. Converts the Interval to a Json string. Be aware that the output might depend on the i18n of your lua interpreter, e.g. if you are using JNLua, that is delivered with the Eclipse Lua Development Tools. This is not taken into account here! You must change the toJson() implementations if you run in such a problem.

Returns:

    A Json string.

Usage:

    Interval:new({v=10,d=0.5}):toJSon()
 will return e.g. '{ "type": "Interval", "value": 10, "low": 9.5, "high": 10.5}'
